QuizHub – Aplicación de Mini-Juegos y Preguntas

Proyecto personal desarrollado en JavaScript, centrado en la creación de distintos mini-juegos interactivos de preguntas y respuestas.
Incluye lógica personalizada para cada juego, diseño responsive y uso de animaciones, puntajes, niveles y feedback inmediato al usuario.


Características principales:

• Múltiples modos de juego (trivia clásica, verdadero/falso, categorías, etc.)

• Sistema de puntajes dinámico y acumulativo

• Interfaz completamente diseñada a mano (HTML + CSS)

• Lógica modular en JavaScript

• Flujo de juego, reinicio, validaciones, temporizadores, animaciones y feedback visual

• Diseño adaptable (responsive)

• Códigos limpios, organizados y pensados para escalabilidad

 Tecnologías utilizadas:

• HTML5

• CSS3

• JavaScript (Vanilla)

• Diseño Responsive

• DOM Manipulation
